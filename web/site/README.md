# site

The primary Astro site for this project.

## Commands

All commands are run from `web/site/`:

| Command           | Action                                       |
| :---------------- | :------------------------------------------- |
| `npm install`     | Install dependencies                         |
| `npm run dev`     | Start local dev server at `localhost:4321`   |
| `npm run check`   | Type-check + diagnostic pass (`astro check`) |
| `npm run build`   | Build to `./dist/`                           |
| `npm run preview` | Preview the production build locally         |

## Standards

Follows `~/.ai/Code.md §10` (accessibility, performance, responsive design, SEO,
privacy). See the root `README.md` for the project-wide layout, and the root
`web/README.md` for the multi-site convention.
